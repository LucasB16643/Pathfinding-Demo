# Pathfinding-Demo

This is a quick and basic demo of some pathfinding algorithms. Just download and run it from an IDE. 

Basic Controls:
- Left Click: place and remove obsticles
- Hold Left Click on Start or End point to drag and move them across the Grid
- 1 key: Swap to using A*
- 2 key: SWap to useing BFS
